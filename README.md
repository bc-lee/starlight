# Starlight

Starlight (Starlark + Light) is an IntelliJ IDEA plugin that adds lightweight Starlark language support for Bazel files.

## Features

- Syntax highlighting
- Line Commenting

## Supported Files

- `BUILD`
- `WORKSPACE`
- `MODULE.bazel`
- `*.bzl`
- `*.bazel`

## Installation

Install from JetBrains Marketplace.

## Development

```bash
./gradlew test
./gradlew verifyPlugin
```
